**TECHNOLOGIES :**
Cette application est réalisée avec les technologies suivantes :
1. Le framework *React* pour le frontend.
2. Le langage de programmation *Python* pour le backend.
3. Le framework python *Django*.
4. L'environnement virtuel *venv*.
5. Le framework de test python *pytest*.
6. L'outil de syntaxe *pylint*.
7. Tout ceci avec une intégration continue sur *GitLab*.
